# 📁 Jayaraya Website — Panduan Folder Assets

Paste foto/video ke folder yang sesuai. Jangan ubah struktur folder.
Nama file boleh bebas, **kecuali** folder yang ditandai ⚠️ (butuh format angka).

---

## ⚠️ Hero/
Hero section background — foto berganti otomatis (looping).
**Penamaan: 01.jpg · 02.jpg · 03.jpg ...**
Minimal: 1 foto | Rekomendasi: 3–5 foto
Format: JPG/WEBP | Resolusi: min 1920×1080

---

## ⚠️ Story/
Film roll animation di section Cerita Jayaraya (panel kanan, scroll ke atas).
**Penamaan: 01.jpg · 02.jpg · 03.jpg ...**
Minimal: 5 foto | Rekomendasi: 8–12 foto
Format: JPG | Rasio bebas (portrait lebih bagus untuk film strip)

---

## ⚠️ Momen/Kiri/ dan Momen/Kanan/
Panel kiri dan kanan section "Semua Momen Ada Disini".
Foto berganti otomatis setiap 1–2 detik.
**Penamaan: 01.jpg · 02.jpg · 03.jpg ...**
Minimal: 3 foto per panel | Rekomendasi: 5–8 foto
Format: JPG | Rasio bebas (portrait oke, landscape oke)

## Momen/Tengah/
Panel tengah — file video Reels yang sudah didownload.
**1 file video saja.** Nama bebas: reels.mp4 / video.mp4 / apapun.
Format: MP4 (H.264) | Durasi: bebas (akan loop otomatis)

---

## Komunitas/
Satu subfolder per event/aktivitas. **Nama file boleh bebas.**
Foto pertama yang dipaste = foto cover (tampil di grid utama).
Semua foto lainnya = galeri dalam modal.

Contoh:
  Komunitas/Standupindo/IMG_7376.jpg      ← cover
  Komunitas/Standupindo/IMG_7377.jpg      ← galeri
  Komunitas/Standupindo/foto_event.jpg    ← galeri

Tambah event baru? Buat folder baru di dalam Komunitas/:
  Komunitas/NamaEventBaru/
